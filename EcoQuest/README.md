# 🌿 EcoQuest — Sustainability App

A cross-platform sustainability awareness app for iOS, Android & Web.
Built with React Native + Expo for BTEC Unit 7: Mobile Apps Development.

---

## ⚡ Quick Start (3 steps)

### Step 1 — Install Node.js
Download from: https://nodejs.org  (choose the LTS version)

### Step 2 — Install dependencies
Open a terminal/command prompt **inside the EcoQuest folder** and run:
```
npm install
```
This downloads all required packages (~1–2 minutes).

### Step 3 — Start the app
```
npx expo start
```

A QR code will appear in the terminal.

---

## 📱 Running on your device

| Platform | How |
|----------|-----|
| **Android phone** | Install "Expo Go" from Play Store → scan the QR code |
| **iPhone** | Install "Expo Go" from App Store → scan the QR code |
| **Web browser** | Press `w` in the terminal after `npx expo start` |
| **Android emulator** | Press `a` (requires Android Studio) |
| **iOS simulator** | Press `i` (requires Xcode on Mac) |
